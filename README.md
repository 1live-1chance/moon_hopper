# Moon Hopper

`1live-1chance - Давид Зиганшин`

Аркадная игра на Python с использованием библиотеки Arcade 3.3.3. Игрок управляет луноходом, собирающим кристаллы на поверхности Луны под действием слабой гравитации, избегая падающих метеоритов.

##  Геймплей
- Собирайте синие кристаллы для набора очков (+10 за штуку)
- Избегайте метеоритов — столкновение заканчивает игру
- 60 секунд на максимальный результат
- Автоматическое сохранение рекорда между сессиями
- Эффекты лунной пыли при прыжках и сборе кристаллов

##  Технологии
- **Python 3.10+**
- **Arcade 3.3.3** — графика, физика, звук, частицы, камера
- ООП-архитектура (9+ классов в отдельных модулях)
- JSON для сохранения рекордов
